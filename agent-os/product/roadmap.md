# Product Roadmap

## Phase 1: MVP

- Quarto blog scaffolding with `_quarto.yml` configuration
- GitHub Actions workflow for publishing to GitHub Pages
- Local F# notebook authoring setup (dotnet-interactive + Jupyter)
- `freeze: true` configuration for pre-computed outputs
- At least one sample blog post as a `.ipynb` notebook

## Phase 2: Post-Launch

- Custom theme and styling
- RSS feed configuration
- Multi-language notebook support (F# and Python in the same blog)
