# Tech Stack

## Publishing

- **Quarto** - Multi-language publishing system for rendering notebooks to HTML

## Notebooks

- **Jupyter** - Notebook authoring environment
- **dotnet-interactive** - Provides the `.net-fsharp` Jupyter kernel for F# execution

## Languages

- **F#** - Primary notebook language
- **Python** - Secondary notebook language (planned)

## CI/CD

- **GitHub Actions** - Automated build and deploy pipeline

## Hosting

- **GitHub Pages** - Static site hosting
